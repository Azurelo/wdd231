Skating Website Plan

Site Name

Skateboard Haven

This name represents a welcoming space for skateboard enthusiasts to find information about skateboarding culture, skateparks, and tricks.

Site Purpose

The site serves as a hub for skateboarding by providing basic information about skating, locations of skateparks, and a curated list of skateboard tricks.

Scenarios

  • Where can I find the nearest skatepark in my area?
  • What are some beginner skateboard tricks I can learn?

Color Schema

Pink (#ff0062) - Used for headings and accents
Gray (#2c2c2c) - Used for body text and backgrounds

Typography

Headings: Arial - Used for all headings (h1, h2, h3)

Body: Georgia - Used for paragraph text
